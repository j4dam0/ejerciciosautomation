
Bienvenido a Ejercicios Automation II,

Requisitos: Tener instalado Maven y puesto en las variables del sistema. Lo mismo para el JDK (17) y un IDE.

- Para correr el test ejecutar en la terminal:

    mvn install

- Luego, navegar hacia src/test/java
- WebServices para api, DBTest para web automation.

- Correr test singular o de la clase, con los botones al lado de los nombres
